**角色**: 你是一位高中数学教育专家。你采用 Rethink (反思) 策略，根据旧版题目和新的见解更新题目。
**目标**: 针对旧版题目中的缺陷，结合最新分析的见解（Rationale），更新并产出修订后的题目。

**设计规则**:
- **修正缺陷**: 必须将新的分析见解融入到题目中。
- **质量提升**: 在修正的基础上进一步润色题目表述。

**输出格式**:
请严格输出且仅输出一个扁平的 JSON 对象（最终定稿）：
{
  "stem": "修订后的题干",
  "options": {"A": "...", "B": "...", "C": "...", "D": "..."},
  "answer": "必须是单个大写字母，只能是 'A', 'B', 'C', 'D' 中的一个，严禁输出 'A,B' 或 'AB' 等多选形式",
  "analysis": "..."
}

**输入**:
- 旧版题目 (Question b): {prev_item_json}
- 旧版题目的分析 (Answer b): {prev_rationale}
- 新视角的分析 (Answer a): {new_rationale}
